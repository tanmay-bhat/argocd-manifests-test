# ArgoCD Reconciliation Lab

Lab to validate `manifest-generate-paths` annotation and `app-resync-jitter` impact on ArgoCD reconciliation. Mimics prod multi-source app pattern.

## Structure

```
├── charts/
│   ├── chart-{1-4}/           # individual charts (Deployment + Service + ConfigMap)
│   │   ├── values.yaml        # base values
│   │   ├── lab-values.yaml    # env-specific override (like prod-us-east-2-values.yaml)
│   │   └── templates/
│   │       ├── deployment.yaml
│   │       ├── service.yaml
│   │       └── cm.yaml
│   └── (bulk-resources/ generated by setup.sh)
├── apps/app-{1-4}.yaml        # individual multi-source ArgoCD Applications
├── argocd-apps/all-apps.yaml  # 350 bulk multi-source Applications (generated)
├── argocd-values.yaml         # ArgoCD Helm chart values (v3.3.4)
├── setup.sh                   # full lab setup (ArgoCD + apps + prometheus)
├── continuous-commit.sh       # pushes commits to chart-1 every N seconds
└── enable-jitter.sh           # patches controller with --app-resync-jitter
```

## Multi-Source App Pattern (matches prod)

Each app has 2 sources:
- **Source 1:** chart path with `$values/` references for valueFiles
- **Source 2:** same repo with `ref: values` (provides the values directory)

```yaml
sources:
  - repoURL: https://github.com/tanmay-bhat/argocd-manifests-test
    targetRevision: HEAD
    path: charts/chart-1
    helm:
      valueFiles:
        - '$values/charts/chart-1/values.yaml'
        - '$values/charts/chart-1/lab-values.yaml'
  - repoURL: https://github.com/tanmay-bhat/argocd-manifests-test
    targetRevision: HEAD
    ref: values
```

## Test Procedure

### Phase 1: Baseline (no annotation, ArgoCD v3.3.4)

```bash
./setup.sh false
# wait for all apps to sync
./continuous-commit.sh 60 10
```

### Phase 2: With annotation + jitter

```bash
./setup.sh true
./enable-jitter.sh
./continuous-commit.sh 60 10
```

## PromQL Queries

```promql
# reconciliation count by app (10m window)
sum by (name) (increase(argocd_app_reconcile_count[10m]))

# total reconciliation rate
sum(rate(argocd_app_reconcile_count[5m]))

# app-controller CPU
sum(rate(container_cpu_usage_seconds_total{pod=~"argocd-application-controller.*"}[5m]))

# repo-server CPU
sum(rate(container_cpu_usage_seconds_total{pod=~"argocd-repo-server.*"}[5m]))

# git requests
sum(rate(argocd_git_request_total[5m]))
```
