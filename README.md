# ArgoCD Reconciliation Lab

Lab to validate `manifest-generate-paths` annotation and `app-resync-jitter` impact on ArgoCD reconciliation.

## Structure

- `charts/chart-{1-4}/` — 4 individual Helm charts (ConfigMap each)
- `bulk-resources/` — shared Helm chart (10 ConfigMaps + 10 Secrets)
- `apps/app-{1-4}.yaml` — 4 individual ArgoCD Applications
- `argocd-apps/all-apps.yaml` — 350 bulk ArgoCD Applications (generated by setup.sh)

## Test Procedure

### Phase 1: Baseline (no annotation)

```bash
# deploy 354 apps WITHOUT manifest-generate-paths
./setup.sh false

# wait for all apps to sync, then start pushing commits to chart-1 only
./continuous-commit.sh 60 10
```

Capture Grafana screenshots after ~10 minutes.

### Phase 2: With annotation + jitter

```bash
# redeploy WITH manifest-generate-paths annotation on all apps
./setup.sh true

# enable jitter on the controller
./enable-jitter.sh

# same commit pattern
./continuous-commit.sh 60 10
```

Capture Grafana screenshots after ~10 minutes.

## Key PromQL Queries

```promql
# total reconciliations per second (expect big drop in phase 2)
sum(rate(argocd_app_reconcile_count[5m]))

# reconciliation duration histogram
histogram_quantile(0.99, sum(rate(argocd_app_reconcile_bucket[5m])) by (le))

# git requests to repo-server
sum(rate(argocd_git_request_total[5m]))

# app-controller CPU
rate(container_cpu_usage_seconds_total{pod=~"argocd-application-controller.*"}[5m])

# repo-server CPU
rate(container_cpu_usage_seconds_total{pod=~"argocd-repo-server.*"}[5m])

# reconciliation count by app (to verify only chart-1 app reconciles in phase 2)
sum by (name) (increase(argocd_app_reconcile_count[10m]))
```

## Scripts

| Script | Purpose |
|--------|---------|
| `setup.sh [true\|false]` | Deploy 354 apps. Pass `true` to enable `manifest-generate-paths` |
| `continuous-commit.sh [interval] [count]` | Push commits to `charts/chart-1/` every N seconds |
| `enable-jitter.sh` | Patch controller with `--app-resync-jitter=60` |
